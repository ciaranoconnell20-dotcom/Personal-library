## Projects

### 🛒 Interactive Shopping List
A fully functional, interactive shopping list application built with HTML, CSS, and JavaScript.

**Features:**
- ✅ Add, check off, and delete items
- 📊 Real-time statistics (total, completed, remaining)
- 🔢 Adjust item quantities
- 💾 Automatic data persistence (localStorage)
- 📥 Export list as text file
- 📱 Fully responsive design
- 🎨 Modern gradient UI with smooth animations

**[👉 OPEN SHOPPING LIST APP](https://htmlpreview.github.io/?https://raw.githubusercontent.com/ciaranoconnell20-dotcom/Personal-library/main/shopping-list.html)**

Or download the file and open it locally:
- [shopping-list.html](shopping-list.html)

---

## How to Use

1. **Open the Shopping List**: Click the link above to use it in your browser
2. **Add Items**: Type an item name and press Enter or click Add
3. **Manage Items**: 
   - Check the box to mark as completed
   - Use +/- buttons to adjust quantities
   - Click × to delete items
4. **Export**: Download your list as a text file
5. **Clear**: Remove all items (with confirmation)
